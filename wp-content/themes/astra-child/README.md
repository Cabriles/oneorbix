# Tema Hijo Astra para OneOrbix

Este es un tema hijo de Astra que contiene personalizaciones específicas para el sitio OneOrbix.

## Características

- **Fix del menú desplegable**: Corrige el problema de texto blanco sobre fondo blanco en el menú desplegable, haciendo que el texto sea visible con color oscuro (#333333).

## Instalación y Activación

1. Ve a **Apariencia > Temas** en el panel de WordPress
2. Busca **Astra Child** en la lista de temas
3. Haz clic en **Activar** para activar el tema hijo
4. Asegúrate de que el tema padre **Astra** esté instalado

## Archivos Personalizados

- `functions.php`: Carga los estilos del tema padre y los estilos personalizados
- `assets/css/custom-menu-fix.css`: Estilos CSS para corregir el menú desplegable

## Notas Importantes

- Este tema hijo preserva todas las funcionalidades del tema padre Astra
- Los cambios personalizados NO se perderán al actualizar el tema padre Astra
- Para desactivar temporalmente las personalizaciones, simplemente activa otro tema

## Mantenimiento

Este tema hijo es independiente del tema padre, por lo que puedes actualizar Astra sin perder tus personalizaciones.
